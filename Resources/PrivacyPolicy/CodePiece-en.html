Privacy Policy

To use this app, you can share a program code text to Gist and Twitter. First, the app post a program code to Gist, then the gist's link and the image capture are posted to Twitter with the description you input. You can also show tweets having hashtags you entered.

The app use twitter account post tweet and display tweets, and use GitHub account only post gist. The app don't use the accounts other purpose.
